    Übersicht¶

    Willkommen auf der Klipper-Dokumentationsseite! Wenn Klipper für dich neu ist, starte doch einfach mit den Features und den Installations-Dokumenten.

    Übersicht¶

    • Features: Eine grobe Übersicht über die Features von Klipper.
    • FAQ: Häufig gestellte Fragen.
    • Releases: Veröffentlichungshistorie von Klipper.
    • Config changes: Die neuesten Änderungen, die potentielle Änderungen an der Konfigurationsdatei des Druckers auslösen können.
    • Contact: Kontaktinformationen bei Fehlern oder Anfragen an die Klipper-Entwickler.

    Installation und Konfiguration¶

    • Installation: Anleitung zur Installation von Klipper.
    • Config Reference: Beschreibung der Einstellmöglichkeiten.
      • Rotation Distance: Berechnung des Schrittmotor-Parameters rotation_distance.
    • Config checks: Überprüfung der grundlegenden Pin-Einstellungen in der Konfigurationsdatei.
    • Bed level: Informationen über "bed leveling" in Klipper.
      • Delta calibrate: Kalibrierung der Delta-Kinematik.
      • Probe calibrate: Kalibrierung von automatischen Z-Sonden.
      • BL-Touch: Einrichten einer "BL-Touch" Z-Sonde.
      • Manuelles Leveln: Kalibrierung der Z-Endstopps (und ähnlichem).
      • Bed Mesh: Betthöhenkorrektion basierend auf XY Koordinaten.
      • Endstop phase: Z-Endstopp-Positionierung mithilfe von Steppermotoren.
    • Resonance compensation: Ein Werkzeug zum reduzieren von Ringing in Drucken.
      • Measuring resonances: Informationen über das Verwenden von adxl345-Beschleunigungssensoren um Resonanz zu messen.
    • Pressure advance: Extruder-Druck kalibrieren.
    • G-Codes: Informationen zu unterstützten Kommandos in Klipper.
    • Command Templates: G-Code Makros und bedingte Evaluierung.
      • Status Reference: Informationen zu verfügbaren Makros (und ähnlichem).
    • TMC Drivers: Trinamic-Steppermotor-Treiber mit Klipper verwenden.
    • Multi-MCU Homing: Homing und Probing mithilfe mehrerer Mikrocontroller.
    • Slicers: "Slicer"-Software für Klipper einrichten.
    • Schräglagenkorrektur: Anpassungen für nicht perfekt rechtwinklige Achsen.
    • PWM tools: Anleitung zum Verwenden von PWM-gesteuerten Werkzeugen (bspw. Laser oder Spindeln).
    • Objekte ausschließen: Die Anleitung zur Objektausschluss-Implementierung.

    Entwicklerdokumentation¶

    • Code overview: Entwickler sollten dies zuerst lesen.
    • Kinematics: Technische Details zur Implementierung von Bewegungen in Klipper.
    • Protocol: Informationen über das low-level Nachrichtenprotokoll zwischen Host und Mikrocontroller.
    • API Server: Informationen über Klipper's Kommando- und Steuerungs-API.
    • MCU commands: Eine Beschreibung von low-level Kommandos, die in der Mikrocontroller-Software implementiert sind.
    • CAN bus protocol: Klipper's CAN-Bus Nachrichtenformat.
    • Debugging: Informationen über das Testen und Debuggen von Klipper.
    • Benchmarks: Informationen über die Klipper Benchmark-Methode.
    • Contributing: Informationen über das Einreichen von Verbesserungen von Klipper.
    • Packaging: Informationen über das Kompilieren von Betriebssystem-Paketen.

    Geräteabhängige Dokumente¶

    • Example configs: Informationen über das Hinzufügen von Beispiel-Konfigurationsdateien zu Klipper.

    • Beaglebone: Details für das Betreiben von Klipper auf der Beaglebone PRU.
    • Bootloaders: Entwicklerinformationen über das Flashen von Mikrocontrollern.
